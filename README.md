## `++i` VS `i++`

下面代码的运行结果是什么？先根据学过的知识计算，再通过编写代码求解。

```java
int x = 1;
int ans1 = x++;
int ans2 = ++x;
int ans3 += (x++) + (++x); // 尽量避免这种代码写法
int ans4 += (x++) + (++x); // 尽量避免这种代码写法
```

## `bitsum` 按位求和

定义函数 `bitsum`，输入一个正整数，输出按位求和的结果。

例如，对于数字 `X = abcd`，定义按位求和操作为 `bitsum(X) = a + b + c + d`。

- `bitsum(123) = 6`
- `bitsum(19260817) = 34`
